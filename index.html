Summary

LendingClubML is a python module for interacting with Lending Club's REST API, and for building and implementing sophisticated investment strategies. This module provides both the basic functionality necessary to programmatically interact with Lending Club's API, and provides a easy framework for implementing machine learning powered investment strategies. The intended users are thus:

1) Investor who prefer to access Lending Club programmatically

2) Scientist and engineers who wish to implement data driven investment strategies

The first group will find everything the need in the LendingClubConnection class. In serving the second group, lending_club_ml.model_lib provides a framework for implementing custom order recommenders.
